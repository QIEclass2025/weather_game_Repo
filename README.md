# 날씨 온도 맞추기 게임

실시간 날씨 데이터를 기반으로 전 세계 주요 도시의 현재 온도를 맞추는 업다운 게임입니다.

## 게임 특징

- 실시간 날씨 API 연동으로 현재 기상 상태 반영
- 전 세계 7개 주요 도시 지원 (서울, 부산, 뉴욕, 런던, 도쿄, 파리, 시드니)
- 현재 날씨 상태, 습도 등 상세 정보 제공
- 소수점 첫째 자리까지의 정확한 온도 맞추기
- 10번의 도전 기회
- 힌트 시스템으로 정답까지의 근접도 표시
- 랜덤 영어 조언으로 재미 요소 추가

## 요구사항

- Python 3.6 이상
- 필요한 패키지:
  - tkinter (GUI 구현)
  - requests (API 통신)

## 설치 방법

1. 저장소를 클론합니다:
   ```
   git clone https://github.com/QIEclass2025/weather_game_Repo.git
   ```
2. 필요한 패키지를 설치합니다:
   ```
   uv 패키지 매니저를 이용해 실행합니다.
    uv run weather_game.py
   ```

## 게임 실행 방법

1. `weather_game.py` 파일을 실행합니다.
2. 드롭다운 메뉴에서 원하는 도시를 선택합니다.
3. 초록색 "🎮 게임 시작" 버튼을 클릭합니다.
4. 입력창에 예상 온도를 소수점 첫째 자리까지 입력합니다. (예: 23.5)
5. Enter 키를 누르거나 "🎯 온도 맞추기" 버튼을 클릭합니다.
6. 힌트를 참고하여 정답을 찾아갑니다.

## 추가 기능

### 조언 받기
- "💡 조언 받기" 버튼을 클릭하면 랜덤한 영어 조언을 받을 수 있습니다.
- 창 하단의 조언 창을 늘려서 전체 내용을 확인할 수 있습니다.
- 게임 플레이 중 언제든 활용 가능합니다.

### 힌트 시스템
- 매우 뜨겁습니다! (±2°C 이내)
- 뜨겁습니다! (±5°C 이내)
- 따뜻합니다! (±10°C 이내)
- 차갑습니다! (±10°C 이상)
