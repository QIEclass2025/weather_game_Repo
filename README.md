날씨 온도 맞추기 게임
====================
소수점 첫째자리까지 맞추기
--------------------

# 실행법
### 온도 맞추기

1. 코드를 실행합니다.
2. 맞추기를 원하는 지역을 선택합니다.
3. "게임 시작"라고 쓰여있는 초록 버튼을 클릭합니다.
4. 온도를 쓰는 칸에 소수점 첫째자리까지 유추해서 씁니다.
5. 엔터키나 밑에 "온도 맞추기"라는 버튼을 클릭합니다.
6. 맞출 때까지 합니다.

### 조언 얻기

1. 조언 받기라는 버튼을 클릭합니다.
2. 마우스로 창의 아래쪽을 늘립니다.
3. 나온 조언을 읽습니다.
4. 깊은 영감을 얻으세요.

## 잘 즐겨주시기 바랍니다. 감사합니다.
